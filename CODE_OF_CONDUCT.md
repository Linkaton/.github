# Code of Conduct

## Our Commitment

Linkaton is committed to providing a friendly, respectful, and welcoming environment for everyone.

## Expected Behavior

Please:

- Be respectful
- Give constructive feedback
- Respect different opinions
- Help others learn and improve

## Unacceptable Behavior

The following are not acceptable:

- Harassment
- Discrimination
- Personal attacks
- Malicious behavior

## Enforcement

Maintainers may remove content or take action against behavior that violates this code of conduct.

Thank you for helping build a positive community.
