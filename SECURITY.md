# Security Policy

## Reporting a Vulnerability

Security is important to Linkaton.

If you discover a security vulnerability, please do not create a public issue.

Instead, report it privately with:

- Description of the vulnerability
- Steps to reproduce
- Possible impact
- Any additional information

## Response Process

We will review security reports and respond as soon as possible.

## Supported Versions

Security updates will focus on actively maintained versions of Linkaton projects.
