## ✨ About Linkaton

Linkaton is an open-source ecosystem focused on building modern tools for link management.

Starting with a powerful URL shortener, Linkaton aims to provide a complete platform for individuals, developers, and businesses to create smarter links.

## 🚀 Projects

| Project | Description |
| --- | --- |
| 🌐 Web | Main Linkaton platform |
| 🔌 API | Developer integrations and services |
| ⚡ CLI | Manage links from the terminal |
| 💻 Desktop | Cross-platform desktop application |
| 📱 Mobile | Android application |
